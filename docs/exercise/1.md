# 练习 1. 规则化路径

## 题目描述

在不少操作系统中，路径可以是不规则的，包括下面几点：

- 表示目录层次的 `/` 和 `\` 可以混用，也可以连用。例如，`/tmp\\foo` 和 `/tmp/foo` 是一样的。
- 可以用 `./` 代表当前目录。
- 可以用 `../` 代表上一层目录。

请编写一个程序，从不规则的路径文本中提取信息，并将每一层路径逐个输出。

## 输入

一行文本，表示不规则的路径，长度不超过 100 字符。

## 输出

输入的路径对应的每层名称，每行一个。

## 样例输入

```plain
tmp///foo/.\bar/../some\else
```

## 样例输出

```plain
tmp
foo
some
else
```
